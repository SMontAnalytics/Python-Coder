Acerca del Desarrollador

Hola, soy Sebastian Montenegro, el creador de este proyecto de concesionario de autos en Django. Este sitio web fue desarrollado como parte de un proyecto final de Python de Coderhouse.

Mi objetivo fue construir una plataforma funcional que simule un concesionario, permitiendo a los usuarios ver listados de autos, detalles específicos, y a los usuarios registrados crear, editar y eliminar entradas de autos, así como gestionar sus propios perfiles.

Si tienes alguna pregunta o sugerencia, no dudes en contactarme.

*Sebastian Montenegro - Coderhouse 2025*
